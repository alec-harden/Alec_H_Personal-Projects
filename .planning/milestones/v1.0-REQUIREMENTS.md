# Requirements: WoodShop Toolbox v1.0

**Defined:** 2026-01-20
**Core Value:** Generate accurate, complete bills of materials for woodworking projects through intelligent guided questioning — reducing planning time and ensuring nothing is forgotten.

## v1 Requirements (All Complete)

Requirements for initial release. Each maps to roadmap phases.

### Platform

- [x] **PLAT-01**: User can view dashboard home page with tool cards — Phase 1

### BOM Generator - Core Flow

- [x] **BOM-01**: User completes guided prompt workflow (project type → dimensions → joinery → materials) — Phase 3
- [x] **BOM-02**: User receives AI-powered material suggestions from configurable LLM provider — Phase 2
- [x] **BOM-03**: User can select project template (table, cabinet, shelf, etc.) to guide AI suggestions — Phase 3
- [x] **BOM-04**: System generates comprehensive BOM covering lumber, hardware, finishes, consumables — Phase 3

### BOM Generator - Editing

- [x] **EDIT-01**: User can edit quantities for any material — Phase 4
- [x] **EDIT-02**: User can add custom materials not suggested by AI — Phase 4
- [x] **EDIT-03**: User can toggle visibility on items (greyed out, excluded from export and totals) — Phase 4
- [x] **EDIT-04**: Materials are grouped by category (Lumber, Hardware, Finishes, Consumables) — Phase 4

### BOM Generator - Export

- [x] **EXPORT-01**: User can export BOM to CSV format — Phase 5
